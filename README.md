## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repo**
```bash
git clone https://github.com/your-username/moshina-portfolio.git

```


2. **Install dependencies**
```bash
npm install

```


3. **Start the development server**
```bash
npm start

```


4. **Build for production**
```bash
npm run build

```
